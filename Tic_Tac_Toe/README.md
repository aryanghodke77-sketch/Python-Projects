# Tic-Tac-Toe Game

A simple Python implementation of the classic Tic-Tac-Toe game where two players can take turns playing. The game board is displayed in the console, and players select their moves by entering numbers from 1 to 9. The game will announce the winner once someone gets three marks in a row or if the game results in a tie.

## Features

- Two-player game (Player 1 and Player 2).
- Players can choose 'X' or 'O'.
- Board updates after each move.
- Winner is announced once a player wins or the game ends in a tie.
- Option to replay the game after completion.

## How to Run

1. Ensure you have Python installed on your system.
2. Download or clone the repository to your local machine.
3. Open a terminal/command prompt and navigate to the project directory.
4. Run the game by executing:

   ```bash
   python tictactoe.py
